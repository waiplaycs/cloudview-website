# 雲向 CLOUDVIEW 網站設計方案

## 方案一：「天際雲境」— 沉浸式天空美學
<response>
<text>
**設計運動**: 天空主義 (Sky Suprematism) — 以天空、雲彩、自然光為核心視覺語言

**核心原則**:
- 天空藍漸層為主調，從深邃的午夜藍過渡至輕盈的霧霾白
- 大量留白模擬高空視野的開闊感
- 文字如雲霧般輕盈飄逸，避免沉重的黑色字體
- 建築圖像以「浮現」方式呈現，彷彿從雲端升起

**色彩哲學**:
- 主色: 天際藍 #87CEEB → 深邃藍 #1B4F72
- 輔色: 雲白 #F0F8FF, 薄霧灰 #D6EAF8
- 點綴: 金色 #D4AC0D (高端質感)
- 情感意圖: 自由、高遠、尊貴

**版面典範**:
- 全屏英雄區域，建築圖像佔據視野75%
- 非對稱文字排版，標題靠左，圖像靠右
- 橫向滾動的設施展示區
- 浮動式導航欄，半透明毛玻璃效果

**標誌元素**:
- 雲朵SVG動畫，緩慢飄移
- 建築輪廓光暈效果
- 波浪形區塊分隔線

**互動哲學**:
- 視差滾動 (Parallax) 增強空間深度
- 滑鼠懸停時圖像輕微放大
- 頁面切換時雲霧淡入效果

**動畫指引**:
- 雲朵以 ease-in-out 緩慢飄移 (8-12秒循環)
- 文字以 fade-up 方式進入視野
- 數據計數器動畫展示樓盤數字

**字體系統**:
- 標題: Noto Serif TC (繁體中文) + Playfair Display (英文)
- 正文: Noto Sans TC
- 強調: 細體 + 大字距
</text>

## 方案二：「翠嶺金廊」— 奢華自然融合
<response>
<text>
**設計運動**: 有機奢華主義 (Organic Luxe) — 自然材質與現代精緻的碰撞

**核心原則**:
- 深綠色 (高爾夫球場) 與金色 (尊貴) 的對話
- 大理石紋理作為背景質感
- 建築圖像與自然景觀並置，強調「城市中的自然」
- 厚重的排版與細膩的留白形成張力

**色彩哲學**:
- 主色: 翠綠 #1A5276 → 深森綠 #0B5345
- 輔色: 香檳金 #F0E68C, 象牙白 #FFFFF0
- 點綴: 玫瑰金 #B76E79
- 情感意圖: 尊貴、自然、私密

**版面典範**:
- 雜誌式排版，圖文交錯
- 全出血圖像配合文字疊加
- 垂直時間軸展示樓盤發展歷程
- 網格式設施展示

**標誌元素**:
- 金色細線框架
- 高爾夫球場俯瞰圖作為背景
- 手寫風格的中文標語

**互動哲學**:
- 滾動觸發的圖像揭示動畫
- 懸停時金色光暈效果
- 流暢的頁面過渡

**動畫指引**:
- 圖像以 clip-path 方式從左至右揭示
- 數字以計數器方式呈現
- 標題以字母逐一出現

**字體系統**:
- 標題: Noto Serif TC + Cormorant Garamond
- 正文: Source Han Sans TC
- 裝飾: 手寫風格字體
</text>

## 方案三：「雲際現代」— 清新都市主義 ✓ 選定方案
<response>
<text>
**設計運動**: 清新都市主義 (Fresh Urbanism) — 以清澈天空藍為靈魂，融合現代簡約與香港都市活力

**核心原則**:
- 天空藍與純白的純淨對比，呼應「雲向」品牌名稱
- 非對稱版面打破傳統地產網站的沉悶感
- 建築圖像以大膽的全屏呈現，強調視覺衝擊力
- 資訊層次清晰，引導用戶自然流覽至查詢表單

**色彩哲學**:
- 主色: 雲天藍 oklch(0.72 0.12 220) — 輕盈、開闊、現代
- 輔色: 純白 oklch(1 0 0) — 潔淨、高端
- 深色: 深海藍 oklch(0.25 0.08 240) — 文字、導航
- 點綴: 薄荷青 oklch(0.85 0.08 185) — CTA按鈕、高亮
- 情感意圖: 清新、自由、值得信賴、現代

**版面典範**:
- 全屏英雄區域，建築圖像右置，文字左置
- 斜切分隔線區分各內容區塊
- 橫向滾動卡片展示設施
- 固定式側邊查詢按鈕

**標誌元素**:
- 雲朵形狀的資訊卡片
- 斜切邊框的圖像展示
- 細線分隔符配合大字距英文

**互動哲學**:
- 視差滾動增強空間感
- 滑鼠懸停時按鈕以天藍色填充
- 數字統計以動態計數呈現

**動畫指引**:
- 英雄區域文字以 stagger fade-up 進入 (0.1s 間隔)
- 圖像卡片以 scale(0.95)→1 的方式進入視野
- 導航欄滾動後背景以毛玻璃效果出現
- 雲朵裝飾元素緩慢漂浮

**字體系統**:
- 標題: Noto Serif TC (繁體中文) + Cormorant Garamond (英文) — 優雅、高端
- 正文: Noto Sans TC — 清晰易讀
- 數字: Playfair Display — 強調數據的重要性
</text>

## 選定方案：方案三「雲際現代」

選擇此方案因為：
1. 天空藍色調完美呼應「雲向」品牌名稱
2. 清新現代風格符合香港高端地產市場定位
3. 非對稱版面避免「AI生成」的千篇一律感
4. 斜切設計元素為網站增添動感與個性
